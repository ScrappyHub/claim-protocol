# CLP Licensing

Claim Protocol (CLP) currently uses the Apache-2.0 license for the protocol text, reference implementation, vectors, and local selftest tooling in this repository.

## Intent

The local deterministic identity layer should remain broadly usable and reproducible so that conformance is possible across implementations.

## Current Boundary

Open in this repository:

- protocol and law text
- reference PowerShell tooling
- vectors
- selftest
- freeze runner

Not yet defined in this repository:

- hosted registry services
- managed witness services
- enterprise indexing / management surfaces
- OEM packaging

Those may be defined later as separate product layers.

## Canonical Truth

This repository currently represents the open deterministic CLP core and its proof surface.
