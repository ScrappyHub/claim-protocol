# CLP Roadmap

## Current Position

CLP is a sealed deterministic identity core.

It is not yet the full standalone instrument.

## Next Phase: Earn Tier-0

To earn Tier-0, CLP must lock:

1. object law
2. media / storage law
3. verifier law
4. licensing / entitlement boundary
5. expanded conformance vectors
6. negative vectors
7. standalone instrument operator surface

## After Tier-0

Only after Tier-0 may CLP integrations begin.

Expected later integrations:

- Echo Transport
- NFL
- WatchTower
- Covenant Gate

Those are later tiers, not current scope.
